## ✨ ¿Qué hace este tema?

**JavieriloxUI Custom** toma la increíble base de *WizardUI* y la transforma en una experiencia visual dinámica. Está diseñado para quienes aman la estética gamer, los fondos de pantalla rotativos y la iluminación RGB, sin sacrificar la legibilidad de la interfaz.

### 🚀 Características Principales

* 🔄 **Rotación de Fondos Integrada (EMB):** El tema cicla automáticamente entre 9 fondos de pantalla de alta resolución (extraídos de Pinterest). La rotación completa dura 9 minutos (540 segundos), dándole vida a tu cliente sin distracciones bruscas.
* 🌈 **Canales No Leídos en RGB:** ¿Mensajes nuevos? El nombre del canal brillará con una **animación de arcoíris fluida** (Rojo, Naranja, Amarillo, Verde, Cian, Azul, Morado, Rosa).
* 💎 **Canal Activo "Celeste Glow":** El canal en el que te encuentras actualmente anula el efecto RGB para mostrarse en un elegante color celeste sólido (`#028df7`) con un efecto de brillo (Glow), para que siempre sepas dónde estás escribiendo.
* 👻 **Interfaz Translúcida:** Menús, listas de servidores y ventanas de chat con fondos oscuros semi-transparentes (`rgba(0,0,0,0.40)`) para que los fondos rotativos siempre sean los protagonistas.
* 🧩 **Addons Premium Incluidos:**
    * Lista de servidores en tonos púrpura.
    * Indicadores de "Usuario escribiendo" mejorados.
    * Animaciones al pasar el cursor (Hover) sobre los miembros.
    * Carpetas de servidores alternativas.

## 🛠️ Instalación

Este tema es compatible con clientes modificados de Discord que soporten inyección de CSS (como **BetterDiscord** o **Vencord**).

### Para BetterDiscord:
1. Descarga el archivo `JavUI-Canales-Celeste-RGB-Chat-20.theme.css`.
2. Abre Discord y ve a **Ajustes de Usuario** ⚙️.
3. Desplázate hacia abajo hasta la sección de BetterDiscord y haz clic en **Temas**.
4. Haz clic en **Abrir carpeta de temas**.
5. Mueve el archivo `.css` descargado a esa carpeta.
6. ¡Activa el tema en Discord y disfruta!

### Para Vencord:
1. Copia todo el código del archivo `.css`.
2. Ve a **Ajustes de Usuario** ⚙️ > **Vencord** > **Temas**.
3. Pega el código en la pestaña de **Temas Personalizados (Custom CSS)** o carga el archivo si usas la versión de escritorio de Vencord.

## 🎨 Personalización (¡Hazlo tuyo!)

El tema está construido con variables de CSS (`:root`) súper fáciles de editar. Si abres el archivo `.css` con cualquier editor de texto, podrás cambiar:

* **Tus propios fondos:** Busca las variables `--EMB_1_HD` a `--EMB_9_HD` y reemplaza los enlaces entre las comillas con tus propias imágenes.
* **Color Principal:** Cambia `--Primary: #028df7;` por tu color hexadecimal favorito.
* **Velocidad de Rotación:** Modifica `--BackgroundTime: 540s;` al tiempo que prefieras.
